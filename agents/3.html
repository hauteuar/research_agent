🎯 What you can analyze:

📚 Copybooks:
  • • Complete field usage across programs
  • • Field creation and modification tracking
  • • Unused/stale field detection
  • • Data transformation analysis
💾 Programs:
  • • Call chains and execution flow
  • • Parameter passing analysis
  • • Data input/output mapping
  • • Business logic extraction
🏷️ Fields:
  • • Creation to deletion lifecycle
  • • Cross-program usage patterns
  • • Validation and business rules
  • • Reference vs update operations
⚙️ JCL Jobs:
  • • Execution flow and dependencies
  • • File processing patterns
  • • Program invocation sequences
  • • Data file lifecycles

🚀 Getting Started:

  1. Upload your COBOL programs, copybooks, and JCL files
  2. Test your vLLM server connection
  3. Enter a component name (suggestions will appear as you type)
  4. Click "Analyze Component Lifecycle" for detailed analysis
  5. Explore different tabs to see field matrices, usage patterns, and dependencies
